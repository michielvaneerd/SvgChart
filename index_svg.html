SvgChart

Small SVG charting library that provides line, bar, pie and donut charts.

Getting started

// First create a new chart with configuration
var myChart = new SvgChart(document.getElementById("myChartDiv"), {
  chartType: 'line',
  series: [
    {
      title: 'Cars',
      id: 'cars'
    },
    {
      title: 'Bikes',
      id: 'bikes'
    }
  ]
});
// And then display the data
myChart.chart({
  series: {
    cars: [12, 45, 67],
    bikes: [33, 32, 23]
  },
  xAxis: {
    columns: ['mon', 'tue', 'wed']
  }
});

Configuration

chartType
Type of the chart, can be: 'line', 'bar', 'lineAndBar', 'pie', 'donut'
series
The series without the values, each serie can be like: 
{
  title: 'Cars',
  id: 'cars',
  color: '#AACC33',
  fillGradient: ['#000000', '#FFFFFF'],
  pointColor: '#000000',
  strokeColor: '#C0C0C0'
}
