SvgChart

SVG Chart library. Below you'll find the documentation. For examples see the examples page.

First create a SvgChart instance by setting a parent DOM element and a config object. Next display the chart by calling the chart() method with the data.

var chart = new SvgChart(document.getElementById(id), {
    chartType: 'line',
    dir: document.documentElement.getAttribute('dir') || 'ltr',
    title: 'My chart',
    minValue: 0,
    maxValue: 100,
    series: [
      {
        id: 'train',
        title: 'Train'
      },
      {
        id: 'car',
        title: 'Car'
      }
    ]
});
chart.chart({
  series: {
    train: [20, 50, 70],
    car: [30, 50, 60]
  },
  xAxis: {
    columns: ['mon', 'tue', 'wed']
  }
});

Configuration

Below you find all possible properties of the config object. This is one object, but below it is separated into separate parts for convenience.

Global

chartType

Type of chart.

One of: line, bar, lineAndBar, pie, donut.

Default: line.

dir

Writing direction.

One of: ltr, rtl.

Default: ltr.

padding
Object that describes the padding.
Default: 
{
  start: 40,
  end: 20,
  top: 100,
  bottom: 40
}
backgroundColor
Background color of the chart.
Default: 'white'
transition
Whether the chart sould be faded into view.
Default: true
fontFamily

Font family

Allowed values are all family families. Examples are: Arial and Verdana

Default: sans-serif

Title

titleFontSize

Font size of label

Default: normal

titleColor

Color of title

Default: black

titleHorizontalPosition

Horizontal position of the title

One of: center, start, end.

Default: center

titleVerticalPosition

Vertical position of the title

One of: center, top, bottom.

Default: top

Axis

Legend

Line and bar charts

Line charts

Bar charts

Pie and donut charts

showValueOnFocus

Whether the value should be displayed in a box after focussing a bar or point on a line.

Default: true

focusedValueFill

Fill color of focused value box

Default: black

focusedValueColor

Color of focused value text

Default: white

maxValue

Maximum value for Y axis

Default: null

minValue

Minimum value for Y axis

Default: null

axisTitleFontSize

Font size of Y and X axis title

Default: smaller

axisLabelFontSize

Font size of Y and X axis labels.

Default: small
